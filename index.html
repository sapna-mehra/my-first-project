🚀 Shelby Docs

📋 System Requirements

  • Node v22 or later
  • Linux or MacOS

📦 Installation

This repo requires Shelby CLI and Aptos installed.

  1. Fork this repository
  2. Clone to local filesystem
  3. Run:
npm install

4. Build project:

npm run build
